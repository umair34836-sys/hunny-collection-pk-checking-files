📺 Video Advertisement Settings

📹 How to Add Advertisement:

  1. Select the page you want to configure from the dropdown below
  2. Upload your video (MP4) or image (JPG/PNG) to assets folder
  3. Select media type below (Video or Image)
  4. Enter the file name (e.g., v1.mp4 or banner.jpg)
  5. Configure settings and click "Save for This Page"
  6. Repeat for each page you want to show ads on

💡 Tip: Each page can have different videos/images. Admin can control which ad shows on which page!
