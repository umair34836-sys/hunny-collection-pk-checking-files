📺 Video Ads Management

⚙️ Configure Ads in Settings

Page-wise Advertisement Configuration

Configure different video/image ads for each page of your website. Each page can have its own ad settings including media type, file, position, and more.
Go to ⚙️ Site Settings to set up ads for each page.

Page Status Media Type File Name Position Actions

📋 Instructions for Setting Up Ads:

  1. Click the Configure button next to any page, or go to Site Settings
  2. Navigate to the Video Ad section
  3. Select the page you want to configure from the dropdown
  4. Toggle Enabled to activate the ad
  5. Upload your media file (video or image) to the assets folder
  6. Enter the file name (e.g., promo.mp4 or banner.jpg)
  7. Choose the position (Top or Bottom of the page)
  8. Optionally add a click URL for users to visit when they click the ad
  9. Click Save to apply changes

💡 Quick Tips:

  • Each page can have a different video or image ad
  • Upload your media files to the assets folder
  • Use MP4 for videos and JPG/PNG for images
  • Keep video files under 5MB for faster loading
  • Use autoplay muted for better user experience
  • Add a click URL to drive traffic to promotions or offers
  • Ads are stored in localStorage per page

⚠️ Important Notes:

  • Video ads are configured per page - each page has independent settings
  • Settings are stored in the browser's localStorage
  • Clearing browser data will remove ad configurations
  • Make sure media files exist in the assets folder or ads won't display
  • Test ads on each page after configuration to ensure they work properly
